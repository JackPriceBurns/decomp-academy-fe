# Load a `u16` at byte offset 12

**`lhz`** (Load Halfword and Zero) loads a 16-bit big-endian halfword from memory into a register, zero-extending it to 32 bits. The instruction encodes a base register and a signed displacement; the CPU reads from `base + displacement`.

For example, given:

```c
typedef struct { u8 _pad[6]; u16 count; } Header;
```

the function:

```c
u16 getCount(Header* h) { return h->count; }
```

compiles to:

```
lhz  r3, 6(r3)
blr
```

The displacement (6) comes from the byte offset of `count` inside `Header`.

## Your task
Write `read` to reproduce the assembly above.

<!-- starter -->
```c
u16 read(S* s) {
    return 0;
}
```

<!-- solution -->
```c
u16 read(S* s) {
    return s->field;
}
```
