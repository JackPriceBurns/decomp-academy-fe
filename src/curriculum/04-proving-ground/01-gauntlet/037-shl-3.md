# Logical shift left by 3

The PowerPC instruction **`slwi rD, rA, n`** (shift left word immediate) is an
extended mnemonic for `rlwinm`. The compiler emits it whenever you left-shift an
unsigned 32-bit value by a constant. The third operand is the shift count.

For example, a function that doubles its argument compiles to:

```asm
slwi    r3,r3,1
blr
```

Inspect the third operand of `slwi` in the target assembly to find the shift
count, then write the corresponding C expression.

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 3;
}
```
