# Multiply by 15

When the source code multiplies by a compile-time constant, MWCC at `-O4` always uses `mulli` — the PowerPC immediate-multiply instruction. The constant is baked into the instruction encoding, making it immediately visible in the disassembly.

For example, multiplying by 30:

```
mulli   r3,r3,30
blr
```

Read the immediate from the target instruction and multiply `x` by that value.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 15;
}
```
