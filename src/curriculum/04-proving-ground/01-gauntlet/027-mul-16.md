# Multiply by 16

When a constant is an exact power of two, MWCC replaces multiplication with `slwi` — shift left word immediate. Left-shifting by `n` is the same as multiplying by 2ⁿ, and the shift amount is encoded directly in the instruction.

For example, multiplying by 32 (= 2⁵):

```
slwi    r3,r3,5
blr
```

Check the shift amount in the target `slwi`. Work out which power of two it corresponds to, then write a multiply expression using that value.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 16;
}
```
