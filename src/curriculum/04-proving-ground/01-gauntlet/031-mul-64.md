# Multiply by 64

When the constant is an exact power of two, MWCC replaces the multiply entirely with a left shift. `slwi` (shift left word immediate) is the extended mnemonic for this. For example, multiplying by 16 — which is 2^4 — emits:

```
slwi    r3,r3,4
blr
```

Look at the target assembly. The shift amount in `slwi` tells you which power of two the compiler chose. Work out that power, then write the corresponding C multiplication.

## Your task
Write `scale` to match the target.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 64;
}
```
