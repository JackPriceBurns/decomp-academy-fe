# Multiply by 9

For non-power-of-two constants, MWCC at `-O4` uses `mulli` — the immediate-multiply instruction — which folds the constant into the encoding rather than using a separate register. The result is always a single instruction regardless of the constant's size.

For example, multiplying by 30:

```
mulli   r3,r3,30
blr
```

The constant you need is sitting in the immediate field of the target `mulli`. Write a simple multiply expression using that value.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 9;
}
```
