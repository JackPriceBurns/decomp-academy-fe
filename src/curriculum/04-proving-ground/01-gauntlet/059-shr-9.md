# Logical shift right

`srwi` is the logical (unsigned) shift-right instruction — it shifts a register right by a constant number of bit positions, filling the vacated high bits with **zero**. Compare it with `srawi`, which is the *arithmetic* (signed) shift-right that replicates the sign bit instead.

For example, a `u32` shifted right by 1 and a `s32` shifted right by 1 look like:

```
halve_flags:
    srwi    r3,r3,1
    blr

halve_signed:
    srawi   r3,r3,1
    blr
```

The immediate encoded in `srwi`/`srawi` is the **shift count**. Read the shift count directly from the instruction in the target assembly — that number is what you shift by in C.

## Your task
Write `shr` on a `u32` to match the target.

<!-- starter -->
```c
u32 shr(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shr(u32 x) {
    return x >> 9;
}
```
