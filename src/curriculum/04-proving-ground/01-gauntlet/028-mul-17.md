# Multiply by 17

When MWCC multiplies an integer by a compile-time constant it never emits a register-form `mullw`. Instead it reaches for `mulli` — the signed immediate multiply. For example, multiplying by 7 produces:

```
mulli   r3,r3,7
blr
```

Look at the target assembly. The `mulli` operand tells you exactly what constant the compiler baked in. Your job is to figure out which constant that is, then write the matching C multiplication.

## Your task
Write `scale` to match the target.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 17;
}
```
