# Multiply by 6

When multiplying an integer by a small constant, MWCC at `-O4` never emits `mullw` (register × register). Instead it uses `mulli` — the immediate-multiply instruction — which encodes the constant directly in the instruction word.

For example, `x * 3` produces:

```
mulli   r3,r3,3
blr
```

The immediate operand tells you exactly which constant is involved. Read the target assembly and identify that constant; a one-operation multiply in C is all you need.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 6;
}
```
