# Multiply by 10

MWCC at `-O4` never emits `mullw` (register × register) when one operand is a compile-time constant. It uses `mulli` instead, encoding the constant as an immediate — giving a compact single-instruction result.

For example, multiplying by 3:

```
mulli   r3,r3,3
blr
```

Inspect the immediate in the target `mulli` to find the constant, then write the corresponding `x * <constant>` expression.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 10;
}
```
