# Multiply by 12

MWCC at `-O4` encodes constant multiplications as `mulli`, not `mullw`. The immediate field in the instruction carries the constant, so there is no need to load it into a register first.

For example, multiplying by 5:

```
mulli   r3,r3,5
blr
```

The constant you need is the immediate operand shown in the target assembly. Write a straightforward multiply expression using it.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 12;
}
```
