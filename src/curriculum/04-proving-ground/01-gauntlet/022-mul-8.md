# Multiply by 8

When the constant is a power of two, MWCC replaces multiplication with a logical left shift — `slwi`. Shifting left by `n` bits is equivalent to multiplying by 2ⁿ, and it is always a single instruction.

For example, multiplying by 4 (= 2²):

```
slwi    r3,r3,2
blr
```

Look at the shift amount in the target `slwi` instruction. Determine which power of two that corresponds to, and write a multiplication by that value.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 8;
}
```
