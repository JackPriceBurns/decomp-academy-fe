# Logical shift left by 15

`slwi` is an extended mnemonic for `rlwinm` that shifts a register left by a
constant amount and fills the vacated low bits with zeros. The shift count is
the immediate field encoded directly in the instruction.

For example, shifting `x` left by 3 produces:

```
slwi    r3,r3,3
blr
```

Read the count from the `slwi` in the target assembly and apply the same
pattern here.

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 15;
}
```
