# Unsigned divide by a power of two

The compiler strength-reduces unsigned division by any power of two into a
logical right shift — **`srwi`** (shift right word immediate). No rounding
fix-up is needed because unsigned right shifts naturally truncate toward zero,
which is what unsigned division requires. (Signed division is different: it
needs an extra correction step.)

The shift count equals log₂ of the divisor. For example,
`quarter_page(u32 x) { return x / 16; }` divides by 2⁴ and compiles to:

```asm
quarter_page:
  srwi   r3, r3, 4
  blr
```

Look at the `srwi` in the target assembly. Its immediate operand tells you the
shift count; work backwards to find the divisor (2^shift), and write the
division expression in C.

## Your task
Write `udiv` on a `u32` so it compiles to the `srwi` above.

<!-- starter -->
```c
u32 udiv(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 udiv(u32 x) {
    return x / 2;
}
```
