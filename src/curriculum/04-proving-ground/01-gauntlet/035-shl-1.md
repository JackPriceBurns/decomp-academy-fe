# Logical shift left by 1

`slwi` (shift left word immediate) is an extended mnemonic built on top of `rlwinm` — it rotates left and masks the low bits to zero. The immediate operand encodes the shift distance. For example, shifting left by 2 produces:

```
slwi    r3,r3,2
blr
```

Look at the target assembly and read the shift distance from the `slwi` operand. Then write the C left-shift expression that uses that same distance.

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 1;
}
```
