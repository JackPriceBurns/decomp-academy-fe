# Logical shift right by 1

`srwi` is an extended mnemonic for `rlwinm` that shifts a register right by a
constant amount, filling the vacated high bits with zeros. This is the
**logical** (unsigned) right shift. On a *signed* value the compiler uses
`srawi` instead, which sign-extends rather than zero-fills.

For example, shifting an unsigned value right by 4 produces:

```
srwi    r3,r3,4
blr
```

Read the shift count from the `srwi` in the target assembly and apply the same
pattern here.

## Your task
Write `shr` on a `u32` to match the target.

<!-- starter -->
```c
u32 shr(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shr(u32 x) {
    return x >> 1;
}
```
