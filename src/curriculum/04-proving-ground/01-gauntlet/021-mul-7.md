# Multiply by 7

When multiplying by a small constant, MWCC at `-O4` reaches for `mulli` — the immediate-multiply instruction — rather than the register form `mullw`. The constant is encoded directly in the instruction, making it straightforward to read off.

For example, multiplying by 5:

```
mulli   r3,r3,5
blr
```

Look at the immediate field in the target `mulli` to learn what constant you need to multiply by, then write the corresponding C expression.

## Your task
Write `scale`, taking one `int`, to reproduce the assembly above.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 7;
}
```
